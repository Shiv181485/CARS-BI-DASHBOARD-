
# 🚗 Cars Analytics Dashboard | Power BI


## Overview
Interactive Power BI dashboard built using the Cars dataset to analyze vehicle pricing, performance, fuel efficiency, manufacturers, and market distribution.

## Features
- Interactive filters
- MSRP analysis
- MPG analysis
- Horsepower & Engine Size analysis
- Origin & Type distribution
- MSRP vs Invoice comparison

## Tools
- Power BI Desktop
- DAX
- Power Query

- Hybrid cars have the best city MPG.
- Europe has the highest combined MSRP.
- Horsepower increases with engine cylinders.
