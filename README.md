# 📖 Project Overview

The **Cars Analytics Dashboard** is an interactive **Power BI** project developed to analyze automobile data and generate meaningful business insights. The dashboard enables users to explore vehicle pricing, engine performance, fuel efficiency, market distribution, and manufacturer trends through dynamic visualizations and interactive filters.

This project demonstrates **data visualization**, **business intelligence**, **data modeling**, **Power Query**, and **DAX** techniques commonly used in real-world analytics projects.

---

# 🎯 Project Objectives

✅ Analyze vehicle prices across manufacturers

✅ Compare MSRP and Invoice prices

✅ Evaluate engine performance

✅ Analyze City & Highway MPG

✅ Compare vehicle origins

✅ Build an interactive business dashboard

# 📈 Key Business Insights

🚗 Mercedes-Benz has the highest total MSRP among manufacturers.

💵 Premium sports cars dominate the highest-priced vehicle segment.

⛽ Hybrid vehicles provide the highest City MPG.

🌍 European vehicles contribute the highest average MSRP.

⚡ Horsepower increases significantly with engine cylinder count.

🚘 Sedan is the most common vehicle type in the dataset.

📊 High-performance vehicles command premium pricing.
