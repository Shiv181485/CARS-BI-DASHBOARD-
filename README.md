
# 🚗 Cars Analytics Dashboard | Power BI

## 📌 Overview
An interactive **Power BI dashboard** built using the Cars dataset to analyze vehicle pricing, engine performance, fuel efficiency, and manufacturer trends. The dashboard provides valuable business insights through dynamic visualizations and interactive filters.

---

## 🎯 Project Objectives
- Analyze car prices across manufacturers and models.
- Compare MSRP and Invoice values.
- Evaluate engine performance and horsepower.
- Analyze fuel efficiency (City & Highway MPG).
- Explore vehicle distribution by origin and type.

---

## 🛠 Tools & Technologies
- Microsoft Power BI Desktop
- DAX
- Power Query
- CSV Dataset

## 📈 Key Insights
- Mercedes-Benz has the highest total MSRP.
- Sedan models dominate the dataset.
- European vehicles have the highest average MSRP.
- Higher cylinder engines generally produce more horsepower.
- Premium sports cars have the highest market prices.
